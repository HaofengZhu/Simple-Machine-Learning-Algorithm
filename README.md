# Simple Machine Learning Algorithm
some simple machine learning algorithm programmed by python
## 文件信息
datasets:存放训练数据，使用CSIE提供的LIBSVM data和IRIS数据集

Linear Regression:使用闭式解和梯度下降算法求解线性回归

Logistic Regression:使用随机梯度下降实现逻辑回归算法

Support Vector Machine:使用随机梯度下降算法实现了支持向量机

KNN:实现的一个KNN类，KNN_test用于测试该类，使用IRIS数据集

Kmeans:实现的一个KNN类，Kmeans_test用于测试该类，使用IRIS数据集
